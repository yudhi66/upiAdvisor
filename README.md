<h1>UPI Advisor - A Crowdsourced Platform to Report Suspicious UPI IDs</h1>

 

 

## 📝 Introduction
The **UPI Advisor** platform allows users to report and retrieve data about suspicious UPI IDs, helping to prevent fraud and ensure secure transactions..   

---

## 🌟 Features

### ✅ User Authentication
- Users can report and comment  on upi id's by registering in the platform.
 

### ✅ Reporting
- Reporting feature for reporting suspicious upi id's
 

### ✅ Report fetching
- Users can fetch full report upi id's.
- Details in report includes: **No of times upi got reported,Risk metrics,Associated Upi id's,Base Upi,Comments on the upi id** .
                           
                             
                           
                            
                               
 

### ✅ Comments
- Authenticate users can comment on their experinces with the upi id.

## 📖 Usage
**Register/Login**: Users must create an account and log in to report UPI IDs or comment on reports.

**Search for a UPI ID**: Use the home page's search bar to check if a UPI ID has been reported.

**View Reports**: Access detailed reports, including risk metrics and user comments.

**Report a Suspicious UPI ID**: Authenticated users can submit reports on potentially fraudulent UPI IDs.

**Engage with the Community**: Users can leave comments and share experiences to warn others.

**Monitor Your Reports**: Users can track the UPI IDs they have reported from their profile section.
