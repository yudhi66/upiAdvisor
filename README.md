## ğŸ“– Usage
**Register/Login**: Users must create an account and log in to report UPI IDs or comment on reports.

**Search for a UPI ID**: Use the home page's search bar to check if a UPI ID has been reported.

**View Reports**: Access detailed reports, including risk metrics and user comments.

**Report a Suspicious UPI ID**: Authenticated users can submit reports on potentially fraudulent UPI IDs.

**Engage with the Community**: Users can leave comments and share experiences to warn others.

**Monitor Your Reports**: Users can track the UPI IDs they have reported from their profile section.
